========================================================================
                     MEOWTEA FRESH - TECH STACK OVERVIEW
========================================================================

1. TỔNG QUAN DỰ ÁN
------------------------------------------------------------------------
- Tên dự án: MeowTea Fresh
- Mục tiêu: Hệ thống đặt đồ uống trực tuyến (Trà sữa, Cà phê, Yogurt...)
- Mô hình: Website thương mại điện tử (B2C)

2. CÔNG NGHỆ BACKEND
------------------------------------------------------------------------
- Ngôn ngữ chính: PHP 7.4+ (Vanilla PHP)
- Kết nối Cơ sở dữ liệu: PDO (PHP Data Objects)
- Kiến trúc:
    + Codebase thuần, không sử dụng framework (Laravel/Symfony).
    + Sử dụng REST API nội bộ để xử lý dữ liệu (JSON format).
    + Quản lý phiên làm việc: Web Sessions.
- Bảo mật:
    + Hash mật khẩu: BCRYPT.
    + Xử lý chuỗi an toàn: htmlspecialchars.
- Modules API (10 nhóm):
    + auth, cart, chatbot, management, menu, news,
      order, product, promotion, stores.

3. CƠ SỞ DỮ LIỆU (DATABASE)
------------------------------------------------------------------------
- Hệ quản trị chính: PostgreSQL (Triển khai trên Supabase).
- Hỗ trợ song song: MySQL / MariaDB (Cho môi trường phát triển local).
- Tính năng:
    + Sử dụng ILIKE cho tìm kiếm không phân biệt hoa thường (Postgres).
    + Cấu trúc dữ liệu chuẩn hóa (Normalized) gồm 19 bảng chính:
        role, appuser, store, user_store, category,
        sanpham, option_group, option_value, product_option_group,
        cart, cart_item, cart_item_option, promotion, payment_method,
        orders, order_item, order_item_option, news, chat_message.
    + Scripts seeding dữ liệu tự động cho Demo.

4. CÔNG NGHỆ FRONTEND
------------------------------------------------------------------------
- Core: HTML5, CSS3, JavaScript (ES6+).
- Thư viện hỗ trợ:
    + jQuery: Xử lý DOM, AJAX, Event handling.
    + Font Awesome / Google Fonts: Icons và Typography.
- CSS Strategy:
    + Vanilla CSS với cấu trúc Modular (chia theo component/page).
    + Sử dụng CSS Variables (cho Theme/Colors).
    + CSS Transitions & Animations cho UX/UI mượt mà.
- Content Management: Tích hợp Markdown (.md) cho các bài viết tin tức.
- Chatbot UI: Giao diện chat nổi (floating), hỗ trợ Markdown,
    tích hợp AJAX để giao tiếp với AI backend qua PHP Proxy.
- Components (14):
    admin-bar, admin-sidebar, back-to-top, button, carousel,
    chatbot, footer, header, modal-box, news-card,
    news-quick-view, pagination, product-card, snack-bar.
- Pages (9 nhóm): about, auth, career, cart, management, menu, news,
    profile, stores.

5. CẤU TRÚC HỆ THỐNG (INFRASTRUCTURE)
------------------------------------------------------------------------
- Web Server: Apache (XAMPP).
- Backend APIs: PHP 7.4+ & FastAPI (Python).
- Database Server: Supabase (PostgreSQL Cloud).
- Media Storage: Local system (assets/img).
- Integration:
    + Tích hợp giả lập thanh toán (MoMo, VNPay, Chuyển khoản).
    + Tích hợp AI Agent cho trải nghiệm mua sắm thông minh.

6. AI CHATBOT SERVICE
------------------------------------------------------------------------
- Ngôn ngữ: Python 3.11+
- Framework: FastAPI 0.111.0 + Uvicorn 0.29.0
- LLM (Large Language Model): Groq — llama-3.1-8b-instant
    (Dự phòng: Google Gemini qua GOOGLE_API_KEY)
- Orchestration: LangChain 0.3.7
    + langchain-groq (kết nối Groq API)
    + langchain-google-genai 2.0.8 (dự phòng Gemini)
    + langchain-community 0.3.7
- HTTP Client: httpx[http2] >=0.24,<0.28
- Quản lý biến môi trường: python-dotenv 1.0.1
- Session Management: In-memory agent pool (1 agent/session),
    tự động tái khởi tạo System Prompt khi user login/logout.
- Chức năng Agent (6 Tools):
    + search_products_tool       — Tìm kiếm sản phẩm theo từ khóa.
    + get_product_details_tool   — Lấy chi tiết + options của sản phẩm.
    + add_to_cart_tool           — Thêm sản phẩm vào giỏ hàng.
    + get_order_status_tool      — Tra cứu đơn hàng theo mã.
    + get_recent_orders_tool     — Xem danh sách đơn hàng gần nhất.
    + search_store_tool          — Tìm cửa hàng theo Quận/Tỉnh.
    (complaint_tool.py — Đã cài đặt, chưa được kích hoạt trong agent.)
- Bảo mật: X-Chatbot-Secret header (CHATBOT_SECRET_KEY).
- Endpoints:
    + GET  /health  — Kiểm tra trạng thái service.
    + POST /chat    — Xử lý tin nhắn và trả về phản hồi AI.
